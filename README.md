# Disease Prediction from Medical Data

## Overview

This project predicts the possibility of diseases using patient medical records and machine learning classification techniques.

## Objective

To develop a machine learning model that predicts whether a patient is likely to have a disease based on medical attributes.

## Dataset

The project uses structured medical data containing features such as:

- Age
- Sex
- Blood Pressure
- Cholesterol
- Blood Sugar
- Heart Rate
- Chest Pain Type

Target:
- Disease Present (1)
- Disease Absent (0)

## Algorithms Used

- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest Classifier

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
